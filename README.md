# AddOn-OpenTrafficCam #
Ein kleiner Zusatz, um die Umbenennungen der Videos und Aktivierung des OTVision-Programms zu automatisieren.
Ebenso können die OTAnalytics-Exporte zur Tabellendarstellung und Verlaufslinie weiterverarbeitet werden.

## Installation ##
Eine Installation wird beim Ausführen der start-Datei durchgeführt. Dabei wird eine venv von Py in dem Projektordner erzeugt. Zusätzlich muss unter den Umgebungsvariablen vom System die Vaiable `OT_PATH` mit Wert = `%Pfad_zum_Ordner_der_OpenTraffic-Programme%` hinterlegt werden. <br>
Unter der folgenden Hierachie muss der Ordner 'OTraffic' als Ziel im Path angegeben werden:
    
    GitHub > OTraffic > OTVision 
					  > OTAnalytics

Ebenso muss R auf dem Rechner installiert und in den Systemumgebung zu finden sein.

## Anwenden ##
Zum Starten des Programms nutzen Sie die `start_gui.cmd`-Datei (mit Fehlermeldungen) oder die `start_gui.vbs`-Datei (ohne Meldungen).
